# Task Manager (Console App)

Tiny Python console app with in-memory tasks and basic CRUD operations.

## Run

```bash
python app.py
```

## Tests

Run tests with:

```bash
python -m pytest -q
```

## Files

- `tasks.py` — Task and minimal `TaskManager` (create, list, get, update, delete)
- `app.py` — Compact console UI
- `tests/test_tasks.py` — Simple CRUD test
